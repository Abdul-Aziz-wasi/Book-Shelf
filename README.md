# 📚 Bookshelf App

A full-featured web application to manage your personal book collection. Users can add, edit, delete, and review books. The app also includes category filtering, searching, upvoting, and user-specific book management. Built with the MERN stack and Firebase authentication for a seamless experience.


## 🚀 Live Link

🔗 [Visit Live Site](https://guileless-gumdrop-8ead0b.netlify.app/bookShelf))

---

## 🛠️ Main Technologies

- **React.js** – Frontend
- **Node.js & Express.js** – Backend
- **MongoDB** – Database
- **Firebase Authentication** – Secure login system
- **Tailwind CSS** – Styling
- **Framer Motion** – Animations

---

## ✨ Main Features

- 🔐 User authentication (sign up, login, logout)
- 📚 Add, edit, and delete books
- 📝 Post and update reviews (1 review per user per book)
- 🔎 Filter books by category or search by title
- 👍 Upvote feature with real-time update
- 📊 Dashboard for viewing user’s books and category-based breakdown
- 📥 "My Books" page – View only books added by the logged-in user
- 📈 Pie chart to visualize book categories
